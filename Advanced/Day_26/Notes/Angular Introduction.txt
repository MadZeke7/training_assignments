2. Environment Setup for Angular
~~~~~~~~~~~~~~~~~~~~~~~~~~~

1.  Install node
		https://nodejs.org/en/
		
	>	node    -v
	>  npm     -v
	
		npm ---  node package manager
		
2.   Install Angular CLI 	(Command Line Interface )

		>  	npm   install   -g     @angular/cli  
		
3.    ng  version

4.   ng  new  my-project

5.  cd  my-project

6.   ng   s  -o
